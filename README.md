# Portpolio 만들기

* 2022.3 ~ 수정 중
* 제가 만들었던 프로젝트들을 정리하여 보여주는 페이지 입니다.
* 1인 제작으로 프로젝트 디자인 구현 및 기획하였습니다.

# 구현 기능

* typing 라이브러리를 사용하여 타이핑효과 구현
* JS로 슬라이드 기능 구현
* Light / Dark 기능 구현
* Navigation 클릭 시 해당 section으로 이동
* 반응형으로 구현

# 시작
```javaScript
$ yarn start
```
